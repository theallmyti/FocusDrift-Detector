# Digital Burnout & Focus Drift Detector

A web application designed to help users track their daily screen habits, analyze focus levels, and detect early signs of digital burnout.

## Features

- **Daily Tracking**: Input screen time, app switches, focus levels, breaks, sleep duration, and work type.
- **Burnout Detection**: Smart algorithm to calculate "Burnout Score" and "Focus Stability".
- **State Analysis**: Detects states like "Focused", "Drifting", or "Burnout Risk" based on user inputs.
- **Visual Analytics**: Interactive charts (using Chart.js) to visualize screen time vs. focus trends and burnout history.
- **Local Storage**: All data is stored locally in your browser for privacy.
- **PWA Support**: Installable as a Progressive Web App (PWA) on mobile and desktop.

## Getting Started

1. Clone the repository.
2. Open `index.html` in your web browser.
3. Start logging your daily digital habits!

## Technologies Used

- HTML5 & CSS3 (Glassmorphism design)
- Vanilla JavaScript
- Chart.js (Data Visualization)
- Service Worker & Manifest (PWA)
